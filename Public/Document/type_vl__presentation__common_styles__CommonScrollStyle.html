CommonScrollStyle Class

A general implementation for GuiScroll::IStyleController to make a scroll control.

Base Types:
GuiScroll::IStyleController

Sub Types:

NameDescription
DirectionScroll direction.

Member Functions:

NameDescription
BuildStyleBuild the style. This function is supposed to be called in the contructor of the sub class.
CommonScrollStyleCreate the style controller using a specified direction.
CreateDecreaseButtonStyleCallback to create a style controller for the decrement button.
CreateHandleButtonStyleCallback to create a style controller for the handle button.
CreateIncreaseButtonStyleCallback to create a style controller for the increment button.
GetBoundsCompositionGet the bounds composition. A bounds composition represents all visible contents of a control. (Inherits from GuiControl::IStyleController.)
GetContainerCompositionGet the container composition. A container composition is where to place all bounds compositions for child controls. (Inherits from GuiControl::IStyleController.)
InstallBackgroundInstall necessary compositions and elements to the background.
SetCommandExecutorCalled when the command executor is changed. (Inherits from GuiScroll::IStyleController.)
SetFocusableCompositionSet the focusable composition. A focusable composition is the composition to be focused when the control is focused. (Inherits from GuiControl::IStyleController.)
SetFontSet the font to render the text. (Inherits from GuiControl::IStyleController.)
SetPageSizeCalled when the page size is changed. (Inherits from GuiScroll::IStyleController.)
SetPositionCalled when the position is changed. (Inherits from GuiScroll::IStyleController.)
SetTextSet the text to display on the control. (Inherits from GuiControl::IStyleController.)
SetTotalSizeCalled when the total size is changed. (Inherits from GuiScroll::IStyleController.)
SetVisuallyEnabledSet the enableing state to affect the rendering of the control. (Inherits from GuiControl::IStyleController.)
