TextLine Class

Represents a line of characters.

Fields:

NameTypeDescription
attCharAtt *A extra information buffer starts from the first character of this line.
availableOffsetCountsigned __int32The number of available CharAtt::rightOffset in the buffer.
bufferLengthsigned __int32The number of elements in the allocated buffer memory.
contextFinalStatesigned __int32The internal context sensitive state of a colorizer when it parses to the end of this line. -1 means that this state is not available.
dataLengthsigned __int32The number of available characters in the buffer.
lexerFinalStatesigned __int32The internal lexical analyzer state of a colorizer when it parses to the end of this line. -1 means that this state is not available.
textwchar_t *A character buffer starts from the first character of this line.

Member Functions:

NameDescription
AppendAndFinalizeAppend a text line after the this text line, and finalize the input text line.
FinalizeRelease all resources used in this line.
InitializeInitialize the TextLine instance to be an empty line.
IsReadyTest is the line initialized.
ModifyModify the characters in the line by replacing characters.
SplitSplit a text line into two by the position. The current line contains characters before this position. This function returns a new text line contains characters after this position.
