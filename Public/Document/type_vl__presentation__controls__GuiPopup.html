GuiPopup Class

Represents a popup window. When the mouse click on other window or the desktop, the popup window will be closed automatically.

Base Types:
GuiWindow

Sub Types:

NameDescription

IStyleControllerStyle controller interface for GuiWindow. (Inherits from GuiWindow.)
IStyleControllerRepresents a style for a control. A style is something like a skin, but contains some default action handlers.(Inherits from GuiControl.)
IStyleProviderA style provider is a callback interface for some control that already provides a style controller, but the controller need callbacks to create sub compositions or handle actions.(Inherits from GuiControl.)
