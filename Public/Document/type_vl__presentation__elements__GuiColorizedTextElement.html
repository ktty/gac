GuiColorizedTextElement Class

Defines a text element with separate color configuration for each character.

Base Types:
IGuiGraphicsElement

Sub Types:

NameDescription
ICallbackAn callback interface. Member functions will be called when colors or fonts of a GuiColorizedTextElement changed.
