GuiGraphicsHost Class

GuiGraphicsHost hosts an GuiWindowComposition in an INativeWindow. The composition will fill the whole window.

Member Functions:

NameDescription
DisconnectCompositionNotify that a composition is going to disconnect from this graphics host. Generally this happens when a composition's parent line changes.
GetAnimationManagerGet the animation manager.
GetCaretPointGet the caret point. A caret point is the position to place the edit box of the activated input method editor.
GetFocusedCompositionGet the focus composition. A focused composition will receive keyboard messages.
GetMainCompositionGet the main GuiWindowComposition. If a window is associated, everything that put into the main composition will be shown in the window.
GetNativeWindowGet the associated window.
GetShortcutKeyManagerGet the IGuiShortcutKeyManager attached with this graphics host.
RenderRender the main composition and all content to the associated window.
SetCaretPointSet the caret point. A caret point is the position to place the edit box of the activated input method editor.
SetFocusSet the focus composition. A focused composition will receive keyboard messages.
SetNativeWindowAssociate a window. A GuiWindowComposition will fill and appear in the window.
SetShortcutKeyManagerAttach or detach the IGuiShortcutKeyManager associated with this graphics host. When this graphics host is disposing, the associated shortcut key manager will be deleted if exists.
