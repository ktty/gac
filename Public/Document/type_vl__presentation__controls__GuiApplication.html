GuiApplication Class

Represents an GacUI application, for window management and asynchronized operation supporting. Use GetApplication to access the instance of this class.

Member Functions:

NameDescription
CloseTooltipClose the tooltip
DelayExecuteDelay execute a specified function with an specified argument asynchronisly.
DelayExecuteInMainThreadDelay execute a specified function with an specified argument in the main thread.
GetExecutableFolderGet the folder of the current executable.
GetExecutablePathGet the file path of the current executable.
GetMainWindowGet the main window.
GetTooltipOwnerGet the tooltip owner. When the tooltip closed, it returns null.
GetWindowGet the GuiWindow instance that the mouse cursor are directly in.
GetWindowsGet all created GuiWindow instances. This contains normal windows, popup windows, menus, or other types of windows that inherits from GuiWindow.
InvokeAsyncInvoke a specified function asynchronously.
InvokeInMainThreadInvoke a specified function in the main thread.
InvokeInMainThreadAndWaitInvoke a specified function in the main thread and wait for the function to complete or timeout.
IsInMainThreadTest is the current thread the main thread for GUI.
RunRun a GuiWindow as the main window and show it. This function can only be called once in the entry point. When the main window is closed or hiden, the Run function will finished, and the application should prepare for finalization.
RunGuiTaskRun the specified function in the main thread. If the caller is in the main thread, then run the specified function directly.
ShowTooltipShow a tooltip.
