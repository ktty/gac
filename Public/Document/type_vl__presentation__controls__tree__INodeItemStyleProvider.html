INodeItemStyleProvider Class

Node item style provider for a GuiVirtualTreeListControl to render visible nodes.

Member Functions:

NameDescription
AttachListControlCalled when a node item style provider in installed to a GuiListControl.
BindItemStyleProviderBind a item style provider. Generally the item style provider will be used to create a item style controller for a list control, because a tree view control is implemented using a list control.
CreateItemStyleCreate a node item style controller from a node item style id.
DestroyItemStyleDestroy a node item style controller.
DetachListControlCalled when a node item style provider in uninstalled from a GuiListControl.
GetBindedItemStyleProviderGet the binded item style provider.
GetItemStyleIdGet a node item style id from a node.
InstallBind a node to a node item style controller.
SetStyleSelectedChange the visual affect of a node item style controller to be selected or unselected.
