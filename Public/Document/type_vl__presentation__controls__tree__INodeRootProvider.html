INodeRootProvider Class

Represents a root node provider.

Member Functions:

NameDescription
AttachCallbackAttach an node provider callback to this node provider.
CanGetNodeByVisibleIndexTest does the provider provided an optimized algorithm to get an instance of a node by the index of all visible nodes. If this function returns true, INodeRootProvider::GetNodeByVisibleIndex can be used.
DetachCallbackDetach an node provider callback from this node provider.
GetNodeByVisibleIndexGet a node by the index in all visible nodes. This requires INodeRootProvider::CanGetNodeByVisibleIndex returning true. If the node is no longer needed, a call to the INodeProvider::Release is needed, unless this is a root node so that its parent is null.
GetRootNodeGet the instance of the root node.
ReleaseViewRelease a requested view.
RequestViewRequest a view for this node provider. If the specified view is not supported, it returns null. If you want to get a view of type IXXX, use IXXX::Identifier as the identifier. When the view object is no longer needed, A call to the INodeRootProvider::ReleaseView is needed.
