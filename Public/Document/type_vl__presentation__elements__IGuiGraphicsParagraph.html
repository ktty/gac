IGuiGraphicsParagraph Class

Represents a paragraph of a layouted rich text content.

Sub Types:

NameDescription
BreakConditionInline object break condition.
InlineObjectPropertiesInline object properties.
TextStyleText style. Items in this enumeration type can be combined.

Member Functions:

NameDescription
GetHeightGet the layouted height of the text. The result depends on rich styled text and the two important properties that can be set using IGuiGraphicsParagraph::SetWrapLine and IGuiGraphicsParagraph::SetMaxWidth.
GetMaxWidthGet the max width for this paragraph. If there is no max width limitation, it returns -1.
GetParagraphAlignmentGet the horizontal alignment for this paragraph.
GetProviderGet the IGuiGraphicsLayoutProvider object that created this paragraph.
GetRenderTargetGet the associated IGuiGraphicsRenderTarget to this paragraph.
GetWrapLineGet if line auto-wrapping is enabled for this paragraph.
HitTestPointFind a visual character by point.
RenderRender the graphics element using a specified bounds.
ResetInlineObjectUnbind all inline objects to a range of text.
SetColorReplace the color within the specified range.
SetFontReplace the font within the specified range.
SetInlineObjectBind an IGuiGraphicsElement to a range of text.
SetInteractionIdReplace the interaction id within the specified range.
SetMaxWidthSet the max width for this paragraph. If the max width is set to -1, the max width limitation will be removed.
SetParagraphAlignmentSet the horizontal alignment for this paragraph.
SetSizeReplace the size within the specified range.
SetStyleReplace the text style within the specified range.
SetWrapLineSet if line auto-wrapping is enabled for this paragraph.
