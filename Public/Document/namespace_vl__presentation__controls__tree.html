NameDescription
IMemoryNodeDataBase type for tree view node data.
INodeItemPrimaryTextViewThe required INodeRootProvider view for NodeItemProvider. This view is always needed to create any customized INodeRootProvider implementation.
INodeItemStyleControllerNode item style controller for a GuiVirtualTreeListControl to render a node.
INodeItemStyleProviderNode item style provider for a GuiVirtualTreeListControl to render visible nodes.
INodeItemViewThe required GuiListControl::IItemProvider view for NodeItemStyleProvider. NodeItemProvider provides this view. In most of the cases, the NodeItemProvider class and this view is not required users to create, or even to touch. GuiVirtualTreeListControl already handled all of this.
INodeProviderRepresents a node.
INodeProviderCallbackCallback object for INodeProvider. A node will invoke this callback to notify any content modification.
INodeRootProviderRepresents a root node provider.
ITreeViewItemViewThe required INodeRootProvider view for TreeViewNodeItemStyleProvider.
MemoryNodeProviderAn in-memory INodeProvider implementation.
MemoryNodeRootProviderAn in-memory INodeRootProvider implementation.
NodeItemProviderThis is a general implementation to convert an INodeRootProvider to a GuiListControl::IItemProvider. It requires the INodeItemPrimaryTextView to provide a GuiListControl::IItemPrimaryTextView.
NodeItemStyleProviderThis is a general implementation to convert an INodeItemStyleProvider to a GuiSelectableListControl::IItemStyleProvider.
NodeRootProviderBaseA general implementation for INodeRootProvider.
TreeViewItemA tree view item. This data structure is used in TreeViewItemRootProvider.
TreeViewItemRootProviderThe default implementation of INodeRootProvider for GuiVirtualTreeView.
TreeViewNodeItemStyleProviderThe default INodeItemStyleProvider implementation for GuiVirtualTreeView.
